#perceptron algorithm

perceptron algorithim

  • Def2.1: The perceptron algorithm performs pattern classification when F is the class of linear threshold functions.
    $$
    (1) Percep = F ={x􏰓→sign(w⊤x):w∈Rd} \
    \begin{align}
    (2) sign(\alpha) &= 1 &w ^{T}x>0\

    &= 0&w ^{T}x=0\

    &= -1&w ^{T}x<0 \

\end{align}
$$

  • Algo2.2:

    ![image-20191009213935259](/Users/huangbenson/Library/Application Support/typora-user-images/image-20191009213935259.png)

  • Theorem2.3: complexity of the algorithm to converge.

    Usage: how even when the data are separable, the Perceptron Algorithm does not necessarily run in time linear to the number of input points. In fact, this results shows us that it is possible to select a small set of points that make δ exponentially small (i.e. xi arbitrarily close to w).

    s.t. If some $w ∈ Rd$ with $ ∥w∥ = 1 $ .correctly classifies all data points$ yi = sign(w⊤xi) ; ∀i = 1, …, n$
    $$
    (1) \text{the algo terminates}\
    (2) \text{terminate time = t} \
    t ^{    } \leq \frac{R ^{2}}{\sigma ^{2}}, R= \max \limits_{i} \left | x {i} \right | ,\sigma = \min \limits{i}w ^{T}x_i y_i
    $$
    Proof:

    ![image-20191009215222828](/Users/huangbenson/Library/Application Support/typora-user-images/image-20191009215222828.png)

    ![image-20191009215255612](/Users/huangbenson/Library/Application Support/typora-user-images/image-20191009215255612.png)

  • Qua2.4: One important feature of this algorithm is that it only uses inner products of the data points. The update step of the algorithm produces a wt which is a linear combination of the xi, so we write it as $wt = 􏰐nj=1 αjxj$ for $αj ∈ R$ and $∥α∥1 = 􏰐nj=1 |αj| ≤ t$. For this choice of $w t$, the prediction rule becomes

  • $$
    \hat{y}i = sign(w_t ^{T}x _{i}) = sign( \sum{j}^{}a _{j}x _{j} ^{T} x _{i})
    $$

    updates become (i is one of the misclassified points)
    $$
    a _{i} ^{(t+1)} = a _{i} ^{(t)} + y _{i}
    $$
    ![image-20191009221728322](/Users/huangbenson/Library/Application Support/typora-user-images/image-20191009221728322.png)

  • Theorm2.5: upper bound for risk of perceptron algorithm

    s.t. $\exist w ∈ R ^{d},∥w∥ = 1,δ > 0,$ we have almost surely $∥X∥ ≤ R$ and$ w ^{T} XY > δ$.
    $$
    fn(X;X1,Y1,…,Xn,Yn) = f _{P,M}(X;X1,Y1,…,XM,YM)\ \text{where M is chosen uniformly from }{1, . . . , n}.\
    EL(fn) ≤ R2
    $$

  • Theorm2.6: better lower bound for risk of perceptron algorithm

    s.t. $\forall f ,δ > 0,R > 0,d > 0$, $\frac{R^2}{\alpha ^2} ≤ min(n,d) $

  • $$
    (1)\exists ; P ;on ; R ^{d} x\left{ \pm1 \right} that ; some ; w \in R^d, \left | w \right | = 1,\
    w ^{T}XY \geq \alpha, \left | X \right | \leq R\
    (2)EL(f _{n})\geq \frac{ \frac{R ^{2}}{\alpha ^{2}}-1}{2n}(1- \frac{1}{n})^{2}
    $$

  • Qua2.11 kernel function

    Usage: 由此看出，即使没有kernel，只要找到映射关系，我们仍然可以实现低维空间映射到高维空间，进而通过高维向量内积运算来寻找最佳分割超平面。但是，kernel的引入，降低了向量内积运算的计算复杂度。无论我们映射到几维空间，我们只要把原有维度代入核函数，就能等价于高维上的内积运算。极端的情况下，特征空间映射到了无穷维度空间，如果没有核函数，根本无法计算出映射后的向量内积。

    （映射到新的特征空间后的内积=原空间核函数）

    https://blog.csdn.net/zhangjun2915/article/details/79261368

    k(xi, xj) is the kernel function of inner product of xi, xj
