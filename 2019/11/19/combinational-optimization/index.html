###backpack problem

  • Def: 0-1 backpack problem

    note:

    • Algo: dynamic programming

      1. define I(k, w ), k means the first k items, w means the capacity of backpack. C*(k, w ) means the optimal of I(k, w)

      2. the propagation equation \[ C*(k, w)= \begin{cases} C*(k-1, w)& wk>w\\ max(C*(k-1, w),\; C*(k-1, w-wk)+pk)& wk \leq w \end{cases} \\ C*(0, w) = 0 \; w = 0...C\\ C*(k, 0) = 0 \; k = 0...n \]

      • Qua: O(nC), pseudo-poly algorithm( if L is bounded by O(n), then c is bounded by O(n), then the DP is polynomial, this is the definition of pseudo-poly)

        the input is n+log2L, L = max(max pj, max wj, C)

    • Algo: dynamic programming 2

      1. define y(k, q), k is the first k items, q meaning the sum of reward, and y(k ,q) is the smallest budget used to reach q. if solution doest exists, y(k ,q ) = inf/null. the optimal of origin question is \(max(q| y(n,q) \leq C)\)

      2. the propagation equation \[ y(k ,q ) = \begin{cases} y(k-1, q)& q<pk\\ min(y(k-1,q), \;y(k-1, q-pk)+wk)&q \geq pk \end{cases} \\ y(k,0) = 0, k=0...n \\ y(0,q) = inf, q=1...P \]

      • Qua: O(nP)

    • Qua: is PTAS class

      proof: 提供一个PTAS算法即可。

      从一般思路出发，设计一个贪婪算法。首先通过密度差异i0得到重量之间的关系，然后同样从密度差异出发，得到密度的关系，把重量转化为价值之间的关系，证明完毕。

    • Qua: is FPTAS class

      proof: 提供一个FPTAS算法即可。

      从一个伪多项式算法出发，设计一个P缩微的实例。（之所以不放C，是因为可行性有可能改变）从这个实例出发，然后在两个问题空间里放缩得到结果。

  • Def: relaxation of backpack problem

    • Algo: greedy

      proof:

      1. if density(i) < density(j), then take j. use delta to prove.

  • Qua: no FPTAS

###take-out problem

  • Def: a special form of shortest path

    • Algo: dynamic programming

      ​ 1) define: start = s, terminal = t, ci = shortest from i to t

      N(i) = neighbor of i, ri = with only 1 order, the best expected output from i to t

      ​ 2) propagation equation \[ rt = r \\ ri = \max \limits_{j \in N(i)} (Pij(2r- cj)+(1-Pij)rj-lij) \]

      ​ 3) use special form of Dij algorithm to solve it （之所以可以用，因为这不仅仅是最长路问题，而是和邻边的max问题；最短路：邻边的min+min关系；最长路，邻边的max+max关系。或者直接加两个负号，-ri = min(-ri+l1)变成最短路问题。
